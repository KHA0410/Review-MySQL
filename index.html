Review MySQL

⭐ MySQL là gì?

- MySQL là một hệ quản trị cơ sở dữ liệu có quan hệ
- SQL (Structured Query Language) là ngôn ngữ truy vấn được dùng để tương tác với cơ sở dữ liệu như MySQL
- DATABASE là nơi lưu toàn bộ dữ liệu của một website (bên trong là các table)

⭐ Cách setup dùng MySQL bằng Docker và TablePlus

- Cài đặt Docker và TablePlus
- Mở cmd và chạy lệnh sau:
docker run --name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 -d mysql
- Mở TablePlus và kết nối với MySQL, với các thông số như sau:
Host: localhost
Port: 3306
User: root
Password: 123456

⭐ Các câu lệnh SQL

TẠO CSDL & Dùng
CREATE DATABASE TÊN;
USE TÊN;

TẠO BẢNG
CREATE TABLE HocSinh(
id INT,
ho_ten VARCHAR(100),
tuoi INT
);

THÊM DỮ KIỆU
INSERT INTO HocSinh (ho_ten, tuoi) VALUES
('Nguyễn Kha', 23);

CẬP NHẬT DỮ LIỆU
UPDATE HocSinh
SET tuoi = 24
WHERE ho_ten = 'Nguyễn Kha';

XÓA DỮ LIỆU
DELETE FROM HocSinh
WHERE id = 1;

TRUY VẤN(Hiển thị bảng)
SELECT * FROM HocSinh; //Hiển thị cả bảng HocSinh

TRUY VẤN CÓ ĐIỀU KIỆN
SELECT * FROM HocSinh
WHERE tuoi = 23; //Chỉ hiện ra các học sinh có tuổi là 23
WHRE ho_ten = '%Nguyễn%'; //Hiển trị ra các học sinh trong tên có chữ Nguyễn

SẮP XẾP VÀ GIỚI HẠN
ASC: tăng dần, DESC: Giam dần
SELECT * FROM HocSinh
ORDER BY id ASC
LIMIT 3; //Chỉ hiển thị 3 học sinh
